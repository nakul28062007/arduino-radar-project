# Arduino Radar Sensor Project

## Overview
An object detection radar system using Arduino, Ultrasonic Sensor, Servo Motor and Processing.

## Components Used
- Arduino Uno
- HC-SR04
- SG90 Servo
- Jumper wires

## Working Principle
Servo rotates ultrasonic sensor.
Sensor measures distance.
Arduino sends data to Processing.
Processing displays radar visualization.

## Features
- 180 degree sweep
- Object detection
- Live radar screen
- Distance visualization

## Setup
1. Upload Arduino code
2. Run Processing code
3. Connect to correct serial port
